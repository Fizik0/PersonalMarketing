TRUE Unit Economics = Revenue - All Hidden Costs

- Реклама (3-25% от оборота)
- Возвраты и брак (5-15% от оборота)
- Штрафы и пени (1-3% от оборота)
- Хранение на складе (0.5-2₽/день/товар)
- Эквайринг (1.5-2.5% от оборота)
Minimum Viable Price расчет:
MVP = (Cost Price + All Ozon Fees + Target Margin) / (1 - Return Rate)
 
Пример:
MVP = (1000₽ + 600₽ + 400₽) / (1 - 0.1) = 2222₽

ФОРМУЛА ИСТИННОЙ СЕБЕСТОИМОСТИ
def calculate_true_unit_economics(price, cost, category, weight, size):
    # Базовые издержки
    ozon_commission = price * get_commission_rate(category)  # 8-45%
    fulfillment = get_fulfillment_cost(weight, size)  # 50-200₽
    packaging = 25  # Средняя упаковка
    
    # Маркетинг
    advertising = price * 0.15  # 15% средние траты
    
    # Скрытые издержки
    returns = price * 0.08  # 8% возвратов
    storage = days_in_stock * 2  # 2₽/день хранение
    acquiring = price * 0.025  # Эквайринг
    
    total_costs = (cost + ozon_commission + fulfillment + 
                  packaging + advertising + returns + 
                  storage + acquiring)
    
    profit_per_unit = price - total_costs
    margin_percentage = (profit_per_unit / price) * 100
    
    return {
        'profit_per_unit': profit_per_unit,
        'margin_percentage': margin_percentage,
        'breakeven_price': total_costs / 0.7  # Мин. цена для 30% маржи
    }
